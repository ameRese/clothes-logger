# Worn!
洋服の着用状況を簡単に記録・管理できるサービスです。  
どの服をいつ着たか、どれだけ活用しているかを可視化して、より効率的なワードローブ管理をサポートします。

## 機能一覧
- ユーザー登録、ログイン機能(Breeze)
- アイテム登録機能
    - アイテム登録、更新、削除
- アイテム絞り込み機能
- 着用記録登録機能
    - 当日の着用登録、解除
    - カレンダーによる着用一括登録、解除
